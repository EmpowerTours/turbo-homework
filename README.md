# TURBO Homework — Cohort 1

Welcome to the **TURBO** weekly homework repo! This is your 52-week journey from zero to Web3 builder.

## How It Works

1. **Fork** this repo to your GitHub account
2. **Complete** each week's assignment in the corresponding `week-XX/` folder
3. **Push** your work — the webhook will auto-verify your submission
4. **Earn** TOURS tokens for each completed week + milestone bonuses

## Curriculum Overview

| Phase | Weeks | Focus | Color |
|-------|-------|-------|-------|
| 1 — Dev Foundations | 1–8 | GitHub, Git, terminal, CI/CD, HTML/CSS/JS | 🔵 |
| 2 — Web3 & Blockchain | 9–20 | Blockchain, Solidity, Foundry, viem, NFTs, DeFi | 🟣 |
| 3 — Full-Stack Dev | 21–36 | React, Next.js, APIs, databases, testing, deployment | 🟡 |
| 4 — Business & Startup | 37–52 | Business model, tokenomics, pitch deck, NITRO prep | 🟢 |

## Rewards

- **100 TOURS** per completed week
- **+500 bonus** at Week 8 (Foundations complete)
- **+1,000 bonus** at Week 20 (Web3 complete)
- **+1,500 bonus** at Week 36 (Full-Stack complete)
- **+5,000 bonus** at Week 52 (Graduation!)
- **Max: 13,200 TOURS** over the full program

## Milestones

Complete all weeks up to a milestone to earn an **on-chain SVG badge NFT**.

## Rules

- Each week has a **required deliverable** file — push it to the correct path
- The webhook checks your push for matching files
- You can work ahead, but rewards are tracked per-week
- **Never commit private keys, .env files, or secrets** to this repo
